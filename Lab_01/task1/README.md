# Lab 1.1: Declarative vs Imperative

Lab 1.1 implements the same highlight toggle UI twice: imperatively (jQuery) and declaratively (React).

## Structure

- **imperative/** — jQuery version (HTML + inline script)
- **declarative/** — React (Vite) version
- **COMPARISON_REPORT.md** — Task 3 written comparison (250–350 words)

## How to Run

### Imperative (jQuery)

1. Open `imperative/index.html` in a web browser (double-click or use a local server).
2. Click "Toggle Highlight" to add/remove the yellow background on the paragraph.

### Declarative (React)

1. `cd declarative`
2. `npm install`
3. `npm run dev`
4. Open the URL shown (e.g. http://localhost:5173)
5. Click "Toggle Highlight" to toggle the highlight.

## Requirements

- **Imperative:** jQuery (loaded via CDN in index.html)
- **Declarative:** Node.js, npm; React 18+, Vite
