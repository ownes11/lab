# Lab 1.2: React Project Setup & Minimal App

Lab 1.2 sets up a React project (Vite) and implements a minimal Counter app.

## Structure

- **src/Counter.jsx** — Counter component with Increment/Decrement
- **src/App.jsx** — Main app that renders Counter
- **VIEW_LAYER_EXPLANATION.md** — Task 3 explanation (200–300 words)

## How to Run

1. `cd Lab_01/task2` (or `cd task2` from Lab_01)
2. `npm install`
3. `npm run dev`
4. Open the URL shown (e.g. http://localhost:5173)

## Project Setup

- **Tool:** Vite (`npm create vite@latest . -- --template react`)
- **Node/npm:** Check with `node -v` and `npm -v`
- **Run:** `npm run dev` (development) or `npm run build` + `npm run preview` (production)
