# Lab 4.1 – Event Handling: Registration Form

This project implements the **Lab 4.1** requirements:

- Controlled inputs for **name**, **email**, and **age** (value + onChange).
- Inline event handlers that use React **synthetic events**.
- Validation on change for all fields with error messages below each input.
- `onSubmit` handler that calls `event.preventDefault()`, validates all fields, and shows a success message when the data is valid.

## How to run

```bash
npm install
npm run dev
```

Then open the local URL that Vite prints in the console.
