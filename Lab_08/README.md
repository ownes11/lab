# Lab 08: Server-Side Rendering with Next.js

Week 8 Assignment: Building a blog application demonstrating SSR and SSG in Next.js.

## Tasks

### Task 1: Next.js Project Setup and Data
- Next.js 14 project with TypeScript
- Type definitions for blog data
- Mock API functions

[Task 1 Details](task1/README.md)

### Task 2: Blog Pages Implementation
- Home page (SSG)
- Individual post pages (SSG with dynamic routes)
- About page (SSG)
- Profile page (SSR)

[Task 2 Details](task2/README.md)

## Technologies Used
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## Running the Projects

Each task is a separate Next.js project:

```bash
cd task1 && npm install && npm run dev
cd task2 && npm install && npm run dev
```
