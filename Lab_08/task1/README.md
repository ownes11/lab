# Lab 8.1: Next.js Blog Setup and Data

This task sets up a Next.js project with TypeScript and creates the basic data structures and mock API for the blog application.

## Features

- Next.js 14 with App Router
- TypeScript configuration
- Tailwind CSS for styling
- Type definitions for Post and Author
- Mock data API functions

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the default Next.js page.

## Project Structure

- `types/index.ts` - TypeScript interfaces
- `lib/api.ts` - Mock data and API functions
